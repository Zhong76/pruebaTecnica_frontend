# Frontend
## Tecnologías
- Angular
- Node.js
- npm

## Versiones utilizadas
-  Angular CLI       : 21.1.5
- Angular           : 21.1.5
- Node.js           : 24.13.0
- Package Manager   : npm 11.6.2

## Instalación:
Ubicarse en el directorio donde se desea descargar el proyecto y ejecutar:
```bash
###Clonar repositorio:
-- git clone https://github.com/Zhong76/pruebaTecnica_frontend.git
###Ubicarse en carpeta:
-- cd frontend-prueba
###Instalar dependencias:
-- npm install
###Ejecución:
-- ng serve

##Abrir en el navegador:
http://localhost:4200
